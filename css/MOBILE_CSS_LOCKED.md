# ⚠️ ВАЖНО: Файл mobile.css ЗАБЛОКИРОВАН для изменений

## Описание

Файл `mobile.css` содержит все мобильные стили для сайта. Этот файл был специально вынесен отдельно, чтобы его не затрагивать при изменении основного файла `style.css`.

## Инструкция

**НЕ ИЗМЕНЯЙТЕ этот файл без явного разрешения!**

Если вам нужно внести изменения в мобильные стили:
1. Свяжитесь с разработчиком
2. Обсудите необходимые изменения
3. Только после явного разрешения вносите изменения

## Дата создания

Файл создан: 2025-01-15

## Содержимое

Файл `mobile.css` содержит:
- Медиа-запросы для устройств с `max-width: 768px`
- Медиа-запросы для устройств с `max-width: 480px`
- Медиа-запросы для устройств с `max-width: 360px`
- Все мобильные адаптации стилей

## Примечание

Все изменения мобильных стилей должны быть сделаны только в этом файле, а не в `style.css` или инлайн стилях в `index.html`.
