📧 Настройка EmailJS для формы записи

Шаг 1: Регистрация в EmailJS

1. Перейдите на emailjs.com

2. Зарегистрируйтесь или войдите в аккаунт

3. Создайте новый сервис (Gmail, Outlook, etc.)

Шаг 2: Создание шаблона

Создайте шаблон с полями:

from_name: {{from_name}} from_phone: {{from_phone}} from_email: {{from_email}} level: {{level}} hall: {{hall}} day: {{day}} message: {{message}}

Шаг 3: Получение ключей

Скопируйте:

  • Service ID
  • Template ID
  • Public Key

Шаг 4: Обновление кода

Замените в файле form-fallback.js:

const EMAILJS_SERVICE_ID = 'ваш_service_id'; const EMAILJS_TEMPLATE_ID = 'ваш_template_id'; const EMAILJS_PUBLIC_KEY = 'ваш_public_key';

Шаг 5: Подключение к HTML

Добавьте в index.html перед закрывающим тегом </body>:

<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script> <script src="form-fallback.js"></script>

⚠️ Важно!

EmailJS бесплатно позволяет отправлять до 200 писем в месяц. Для большего объема нужен платный план.
